SiYuan là một hệ thống quản lý tri thức cá nhân (PKM) mã nguồn mở, tập
trung vào privacy + self-host + offline-first.

🔥 SiYuan là gì?

\* App ghi chú + quản lý kiến thức

\* Chạy local, có thể self-host

\* Hỗ trợ sync mã hóa end-to-end ￼

👉 Hiểu đơn giản:

một “second brain” nhưng bạn kiểm soát toàn bộ dữ liệu

⚡ Điểm nổi bật

\* Block-based editor (giống Notion)

\* Bidirectional links (giống Obsidian)

\* Hỗ trợ outline, graph, mindmap

\* Markdown + WYSIWYG

\* Query bằng SQL trên notes

\* Xử lý tài liệu lớn (hàng triệu từ) ￼

🔐 Privacy-first (điểm khác biệt lớn)

\* Dữ liệu lưu local, không phụ thuộc cloud

\* Có thể dùng offline hoàn toàn

\* Không thu thập dữ liệu người dùng ￼

👉 Phù hợp ai cần:

\* bảo mật

\* kiểm soát dữ liệu

\* không lock-in nền tảng

⚙️ Tech & hệ sinh thái

\* Viết bằng TypeScript + Go ￼

\* Có app đa nền tảng (Windows, Mac, Linux, mobile) ￼

\* Có API → build automation / agent

💡 Use case

\* Second brain / knowledge base cá nhân

\* Quản lý tài liệu dev, research

\* Ghi chú học tập, viết lách

\* Xây hệ thống note tự host
